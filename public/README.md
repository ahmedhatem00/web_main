# 🛍️ نظام المتجر الإلكتروني مع الدعم الفني

نظام متكامل لإدارة متجر إلكتروني مع دردشة مباشرة مدعومة ببوت تليجرام.

## 📁 بنية المجلدات

```
project/
├── server/               ← كود الخادم (Node.js)
│   ├── server.js         ← نقطة الدخول الرئيسية
│   ├── bot.js            ← بوت تليجرام
│   ├── websocket.js      ← اتصال WebSocket
│   ├── sessions.js       ← إدارة الجلسات
│   ├── storage.js        ← تخزين البيانات (JSON)
│   ├── tunnel.js         ← النفق العام
│   ├── github-updater.js ← تحديث GitHub تلقائياً
│   ├── package.json
│   ├── .env.example      ← نموذج متغيرات البيئة
│   └── data/             ← ملفات البيانات (تُنشأ تلقائياً)
│
├── public/               ← الواجهة الأمامية (GitHub Pages)
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── config.js         ← يُحدَّث تلقائياً برابط السيرفر
│
└── .github/workflows/
    └── update-url.yml    ← GitHub Action لتحديث الرابط
```

## 🚀 التثبيت والتشغيل

### 1. إعداد الخادم

```bash
cd server
npm install
cp .env.example .env
# عدّل .env وأضف البيانات المطلوبة
npm start
```

### 2. متغيرات البيئة (.env)

| المتغير | الوصف | مطلوب |
|---------|-------|-------|
| `BOT_TOKEN` | توكن بوت تليجرام | ✅ |
| `ADMIN_CHAT_ID` | Chat ID الخاص بك في تليجرام | ✅ |
| `ACCESS_TOKEN` | GitHub Personal Access Token | اختياري |
| `GITHUB_OWNER` | اسم المستخدم على GitHub | اختياري |
| `GITHUB_REPO` | اسم المستودع | اختياري |
| `PORT` | المنفذ (افتراضي: 3000) | اختياري |
| `TUNNEL_ENABLED` | تفعيل النفق (افتراضي: true) | اختياري |
| `PELLA_URL` | رابط pella.app إن وُجد | اختياري |
| `BASE_URL` | رابط ثابت للخادم | اختياري |

### 3. الاستخدام مع pella.app

إذا كنت تستضيف الخادم على pella.app:

```env
PELLA_URL=https://your-app.pella.app
TUNNEL_ENABLED=false
```

سيُرسل البوت الرابط للمدير تلقائياً عند التشغيل.

### 4. إعداد GitHub Pages

1. ارفع مجلد `public/` على مستودع GitHub
2. فعّل GitHub Pages من إعدادات المستودع
3. أضف `ACCESS_TOKEN` كـ Secret في المستودع
4. عند تشغيل الخادم، يتم تحديث `config.js` تلقائياً

## 🤖 أوامر البوت

| الأمر/الزر | الوظيفة |
|------------|---------|
| `/start` | القائمة الرئيسية |
| `/stats` | إحصائيات المتجر والجلسات |
| `/getlink` | الرابط الحالي للخادم |
| 📦 إدارة المنتجات | إضافة/تعديل/حذف المنتجات |
| 🏷️ إدارة الفئات | إضافة/حذف الفئات |
| 💬 الدردشة | حالة الجلسات النشطة |
| رد على رسالة عميل | إرسال الرد للعميل مباشرة |
