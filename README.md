# نظام الدعم الفني والمتجر الإلكتروني

نظام متكامل لإدارة متجر إلكتروني مع دعم مباشر عبر بوت تليجرام.

## المميزات

- ✅ عرض المنتجات مقسمة حسب الفئات
- ✅ دردشة مباشرة مع الدعم الفني
- ✅ حفظ جلسات المستخدمين عبر الكوكيز
- ✅ إدارة كاملة للمنتجات والفئات عبر بوت تليجرام
- ✅ إنشاء رابط عام تلقائي (localtunnel)
- ✅ إرسال رابط الخادم للمدير تلقائياً
- ✅ دعم إرسال الصور في الدردشة
- ✅ حذف تلقائي للجلسات غير النشطة
